# BillGuardian-Pro

## AI-Based Hospital Billing Fraud Detection System

### 📌 Overview
BillGuardian-Pro is an intelligent system designed to analyze hospital bills and identify potential overcharging, duplicate entries, and suspicious billing practices. It uses OCR (Optical Character Recognition) to extract text from bill images and performs analysis to improve transparency in medical billing.

---

### 🚀 Features
- Extracts bill data using OCR
- Detects duplicate charges
- Identifies unusually high pricing
- Highlights suspicious billing entries
- Simple and user-friendly interface

---

### 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OCR (Optical Character Recognition)

---

### 💡 How It Works
1. User uploads a hospital bill (image)
2. OCR extracts text from the bill
3. System analyzes billing entries
4. Flags duplicate or suspicious charges
5. Displays results clearly

---

### 🎯 Objective
To bring transparency and fairness to hospital billing systems using intelligent analysis and OCR technology.
